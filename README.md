### Задание 2 по дисциплине "Аналитика данных" - API-пайплайн

<b>Задача:</b> данный скрипт должен читать отзывы из входного файла (параметры ниже) и отправлять их в LLM для определения тональности и тематики. Ответ LLM должен обрабатываться и записываться в CSV.
<i>[Важно] Скрипт работает с LLM с помощью API OpenRouter.</i>

<b>Категории тональности, которые определяет скрипт:</b>

- positive
- negative
- neutral

<b>Требования к входному файлу:</b>

- Файл должен быть в формате CSV.
- Должен иметь следующую структуру:

```
id,review
1,"Телефон держит заряд двое суток, экран яркий и сочный."
2,"Ждал курьера 2 часа, в итоге заказ отменили."
3,"Обычная доставка, всё как обычно."
```

#### Запуск скрипта:

Для запуска скрипта нужно иметь установленные библиотеки:

- pandas
- os
- json
- openai
- dotenv

Также нужно иметь файл .env (переименовать файл .env.example) в одной папке со скриптом. В данном файле нужно вставить свой ключ для доступа к API.

Чтобы скрипт заработал, запустите файл script.py.

В результате работы будет получен следующий файл:

```
id,sentiment,theme
1,positive,качество
2,negative,сервис и доставка
3,neutral,сервис и доставка
```
